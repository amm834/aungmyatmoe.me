Enterprise Level React Application တွေက Modular Monolithic မဟုတ်ရင်၊ Micro Frontend ဖြစ်သင့်တယ်။ Interactive ဖြစ်တဲ့ State တွေက Server Side ပေါ်မှာတင်ထားသင့်တယ်။ SSC လိုမျိုး။ Modular Monolithic ဖြစ်တဲ့အခါမှာလည်း FDD ကိုသုံးသင့်တယ်လို့ယူဆတယ်။
မဟုတ်ရင် Agile သုံးပြီးတော့ Sprint Goals မမှီတာတို့ဘာတို့ဖြစ်တာမျိုးပေါ့။ ဒါတွေကို Prevent ဖို့က ဘယ်ကောင်ကိုဘယ်အချိန်မှာဘယ်လို Apply လုပ်သင့်တယ်ဆိုတာသိသင့်ပါတယ်။ ဒါမျိုးက Management Level နဲ့ Technical Level က Catch Up မဖြစ်တာမျိုး၊ သေချာနားမလည်တာမျိုးဆိုရင်ဖြစ်တတ်တယ်။
Modulars Monolithic သုံးတဲ့နေရာမှာလည်းသူ့ Feature နဲ့သူထားပြီးလုပ်ဖို့လိုအပ်တယ်။ ဒီနေရာမှာသိရမှာ Agile Culture ခုမှစပြောင်းတာလား၊ အစကတည်းကရှိပြီးသားလားဆိုတာကလည်းမူတည်တယ်လို့ယူဆနိုင်ပါတယ်။ Shared Package တွေကိုဘယ်လို Management လုပ်မှာလဲ၊ ဘယ်လိုမျိုး ထားသင့်သလဲ၊ Versioning ကအစသိထားသင့်တယ်။
